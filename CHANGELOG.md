# Changelog

All notable changes to the "syntask" extension will be documented in this file.

## [0.0.1] - 2026-06-25

### Added

- Hover explanations for 45+ C# keywords and modifiers
- Context-aware content (e.g., `abstract` on class vs method)
- Semantic analysis for `var` (type inference), `default`, `nameof`, `typeof`, `dynamic`
- Configurable detail level: beginner, intermediate, advanced
- Toggle examples on/off
- Command palette: "C#: Explain Syntax at Cursor"
- Hover-only or command-only trigger modes
